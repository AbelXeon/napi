# Jonathan's Portfolio: A Visual Storytelling Experience

This is the official repository for the personal portfolio website of Jonathan, a passionate visual storyteller. This site showcases a collection of moments captured with soul and perspective, presented in a clean, modern, and engaging interface.

## ✨ Features

*   **Dynamic Image Slider:** The homepage features a stunning full-page slider that highlights key visual stories.
*   **Interactive Gallery:** A dedicated "Work" page to browse through an extensive collection of photographs.
*   **Seamless Navigation:** A clean and intuitive navigation bar that is loaded dynamically across all pages.
*   **Responsive Design:** The website is designed to provide an optimal viewing experience across a wide range of devices.

## 💻 Technologies Used

*   **HTML5:** For the structure and content of the website.
*   **CSS3:** For styling and creating a visually appealing layout.
*   **JavaScript:** For dynamic features like the image slider and component loading.

## 🚀 Getting Started

To view the website locally, follow these steps:

1.  **Get the code:**
    Download the code as a ZIP file or clone the repository to your local machine.
2.  **Navigate to the project directory:**
    Unzip the file if you downloaded it, and then open the project folder.
3.  **Open `index.html` in your browser:**
    You can do this by double-clicking the `index.html` file in the project directory.
